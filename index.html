Học JavaScript Từ Cơ Bản Đến Nâng Cao

JavaScript là ngôn ngữ lập trình phổ biến nhất cho phát triển web. Nó giúp website trở nên động, tương tác và mạnh mẽ.

JavaScript là gì?

JavaScript được tạo ra năm 1995 bởi Brendan Eich. Ngày nay JavaScript được sử dụng trong Front-End, Back-End, Game Development và nhiều lĩnh vực khác.

Biến

let name = "Đại"; const age = 20; console.log(name);

Câu lệnh điều kiện

let age = 20; if(age >= 18){ console.log("Người lớn"); }

Vòng lặp For

for(let i=0;i<5;i++){ console.log(i); }

Function

function add(a,b){ return a+b; }

Array

let fruits = [ "Táo", "Cam", "Xoài" ];

Object

let user = { name:"Đại", age:20 };

DOM

document.getElementById("demo") .innerHTML = "Hello";

Async / Await
