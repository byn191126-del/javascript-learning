JavaScript là gì?

JavaScript là ngôn ngữ lập trình được sử dụng để tạo các trang web động và tương tác. Nó là một trong ba công nghệ cốt lõi của World Wide Web cùng với HTML và CSS.

Ngày nay JavaScript được sử dụng trong:

  • Website
  • Ứng dụng web
  • Máy chủ
  • Game
  • Ứng dụng di động
